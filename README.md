**Boost PageSpeed, reduce your carbon footprint, and track engagement.** Eco-Shield is a high-performance WordPress plugin that replaces heavy YouTube & Vimeo embeds with smart, privacy-focused static players. Standard embeds can load over 1.7MB of scripts—Eco-Shield reduces this to nearly zero until the user clicks "Play."

---

## The "Engagement Update" (v1.3.0)
I engineered several advanced systems into this latest version to move beyond simple performance:
* **Cinema Mode:** Implemented a beautiful, dimmed Lightbox popup overlay for distraction-free viewing.
* **Engagement Analytics:** Built a custom tracking system to see "Play" counts directly in the WordPress Dashboard.
* **Custom Branding:** Added a native Color Picker so users can match the "Play" button to their brand.
* **Carbon Calculator:** Real-time tracking of MBs saved and CO2 reduction.

## Key Technical Features
* **Zero Bloat:** No jQuery. Built with <10KB of modern Vanilla JS.
* **WebP Optimization:** Automatically converts video thumbnails to WebP to save bandwidth.
* **GDPR/DSGVO Ready:** No 3rd-party connections are made until the user interacts.
* **YouTube Shorts Support:** Native detection and 9:16 vertical player rendering.
* **Smart Fallbacks:** Automatic static image conversion for RSS feeds and HTML emails (Gmail/Outlook).

## Installation
1. Search for **Eco-Shield** in your WordPress Dashboard.
2. Or download from the [Official WordPress Plugin Directory](https://wordpress.org/plugins/eco-shield/).
3. Activate and visit **Settings > Eco-Shield** to customize.
